03 · Project

Featured Build

two-tier-flask-app / README.md

Two-Tier Flask + MySQL Application

A containerized two-tier web application with a Flask front end and a MySQL back end, deployed on an AWS EC2 instance. Docker Compose orchestrates both containers over a custom bridge network with persistent volumes, so data survives container restarts. Jenkins watches the GitHub repository and automatically builds and redeploys the app whenever new code is pushed, turning what used to be a manual deploy into a single git push.
